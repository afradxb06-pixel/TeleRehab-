TeleRehab is a better way to rehabilitate patients. It not only provides an easy interactive system for 
patients to rehabilitate more easily but also helps track their progress for physiotherapists.
Existing applications have hard to read and complicated systems, especially for patients. 
They also either don’t have proper support for different platforms or aren’t properly 
supported. To counter these issues, we will develop an easy to use and accessible application. 
The application will also have proper tracking, clear graphs and will meet the local safety requirements.
